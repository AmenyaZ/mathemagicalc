Algorithm for Solving an IVP using the Euler's forward method

This algorithm is valid for any Initial Value Problem together with it's Initial condition

Given an Initial Value Problem together with the step size (h), interval $[x_{0}, x_{n}]$ where one is required to determine the number of times it will take to iterate the function, that is $[h + h]$ until $h > x_{n}$
Substitute the given function, $[f(x, y)]$ into the formula $Yn+hf(Xn, Yn)$, where $X,Y$ will take the values in the function defined with the current value of $n$.
Iterate the function until you get the number of iterations generated at step 1.
Note: At each level of iteration, the value of $h$ will be the nth iterate. For eg. during the 2nd iteration, the value of $h$ will be the second step size generated when $[h+h]$ was performed.
