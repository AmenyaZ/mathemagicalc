Algorithm for converting any binary number to a decimal number

This algorithm is valid for any binary number (n) where n can either be a whole number or a fraction

Given a binary number(N), we raise each digit to a power of it's placeholder.
Given N = 10102, we obtain
$$1^{3}0^{2}1^{1}0^{0}$$
The individual digits are multiplied by 2n where n is the placeholder value.
The products are summed to obtain the decimal numeral of the binary number.
In the case of 1010, we obtain
$$(1\times2^{3}) + (0\times2^{2})+(1\times2^{1}) +(0\times2^{0})$$ $$(8+0+2+0)$$ $$10$$
