Algorithm for approximating the root of an equation using the Bisection Method

This algorithm is valid for equations of the form f(x) = 0

Given that the root of a function f(x) lies in a given interval say [x0, x1]
,
Compute f(x0) and f(x1).
Case 1
If f(x0)f(x1) = 0, then the objective of the algorithm has been fulfilled. That is, one of the end points of the interval x0 or x1 is the root of f(x).
Case 2
If f(x0)f(x1) < 0, then the root actually lies in [x0, x1]
Proceed to step 3
Bisect the interval [x0, x1] to get the midpoint x1 using the formula $$x^{1}=\frac{x_{0}+x_{1}}{2}$$
Case 1
if f(x0)f(x1)<0, then the root lies in [x0, x1], Discard the interval [x1, x1].
Take [x0, x1 as the new interval and set x1 = x1
Case 2
If f(x1)f(x1)<0, then the root lies in [x1, x1].
Discard the interval [x0, x1] and take [x1, x1] as the new interval by setting x0 = x1
if $\frac{\left | x_{1}-x_{0}\right |}{2} < tol$,
then take $x^{*} = \frac{x_{1}+x_{0}}{2}$
STOP: else go back to step 3
