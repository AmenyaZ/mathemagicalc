Algorithm for approximating the root of an equation using the Newton Raphson Method

This algorithm is valid for equations of the form f(x) = 0

Given that the root of a function f(x) lies at a point $[x_{0}]$

Compute $$f_{0} = f(x_{0})$$
Where $x_{0}$ is the initial guess
compute $f^{1}(x_{0})$ i.e is the first derivate of the function $f(x_{0})$
Using the formula, $$x_{1} = x_{0} - \frac{f(x_{0})}{f^1(x_{0})}$$
Replace $x_{0}$ with $x_{1}$ and go back to STEP 1
STOP if two successive iterates are equal
else
Go to STEP 2
