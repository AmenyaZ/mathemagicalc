Algorithm for converting a decimal fraction to a binary number

This algorithm is valid only for a number (n) where n is an fraction, that is anumber greater than or equal to one

Given a decimal number(N), let N1 be the number to convert. That is, let N1 = N
Solve for 2N1 by multiplying N1 by two (2) and keep the results.
If 2N1 is greater than or equal to one (1) i.e (2N1 >= 1), take b-1 = 1
If 2N1 is less than one (1), take b-1 = 0
Using the information gathered in steps 1, 2 and 3, the binary number can be computed using the iterative equation
$$N_{k+1} = 2Nk - b_{-k}$$ The value obtained is reused in subsequent iterations until the stopping criteria is met i.e Nk becomes 0
The final binary equivalent of the decimal number is read in terms of b-k in the order $$b_{-1}b_{-2}...b_{-k}$$ where k is the number of iterations performed in step 4
